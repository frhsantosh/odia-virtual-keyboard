Odia Virtual Keyboard

Type Odia text by clicking on the keyboard keys below.

  1. Typing basic letters: Click a letter and it appears in the text box.
  2. Using matras: Type consonant first, then click matra.
    Example: କ + ା = କା, କ + ି = କି
  3. Using Halant for conjunct letters: Click first consonant, then Halant (୍), then second consonant.
    Example: ଙ + ୍ + କ = ଙ୍କ, ତ + ୍ + ର = ତ୍ର
  4. Using Purnachhed: Use । to end a sentence.
